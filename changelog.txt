Version: 2.1.0
Date: 10. 03. 2023
  Features:
    - New tech: Liquid fuels, requiring flammables and advanced oil processing which unlocks:
    - New building: Petrochemical plant, a specialized plant for fuel production and cracking
    - New building: Gas power station
    - New building: Gas furnace
    - Liquid fuel: a burnable fuel with better fuel value. You should use it, but can technically use any fluid with mixed results
  Adjustments:
    - Solid rocket fuel now uses liquid fuel instead of light gasoil
    - Liquid fuel is now a tech prerequisite for rocket fuel
    - Migration added. Please report any issues / broken saves / permalocked recipes, etc.
    - Coal liquefaction was re-introduced with version 2.0.0 but was undocumented
  Minor:
    - Heavy oil graphics updated, as well as dependent graphics like oil processing
    - Polyethylene grapics updated
